Java Class

Posted on 2022-04-13 In Programming Language , Java

1. 제어자

클래스, 필드, 메서드, 생성자 등에게 특징을 부여하는 문법

1.1 Access modifier

  • private < package(생략가능) < protected < public
    • private: 자기 클래스내에서만 접근 가능
    • package : 같은 패키지 안에서만 접근 가능
    • protected : 다른 패키지일지라도 상속관계가 있으면 접근 가능
    • public : 어디서나 접근 가능
  • 대상 = {필드, 메서드, 생성자} + 클래스
    • 클래스의 경우 package와 public을 구분해서 쓴다.
    • 다른 패키지에서 임포트를 허용할 클래스라면, public을 써야만한다.

Posted on 2022-04-13 Edited on 2022-05-26 In DB/SQL

SQL : RDBMS의 데이터를 제어하기 위한 표준 질의어

NUMBER

  • BINARY_INTEGER : (4Byte:-2^31-1 ~ 2^31-1)
  • NUMBER : 38자리의 정수,실수
  • NUMBER(n) : n자리의 정수
  • NUMBER(p,s) : p자리의 실수, 소수점은 s자리 (소수점이 길어지면 반올림 처리)

CHAR

  • CHAR(n) : n(<2000) 바이트 “고정” 문자열
  • NCHAR(n) : n(<1000) 길이의 문자열
  • VARCHAR2(n) : 가변 길이의 문자열, 최대 n(<4000) 바이트
  • NVARCHAR2(n) : 가변 길이의 문자열, 최대 n(<2000) 길이
  • LONG : 가변 길이의 문자열, 최대 4GB
